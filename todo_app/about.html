About Task List 2023 with LocalStorage

This is a simple to-do list application that allows you to create, categorize, and manage your tasks. It uses local storage to save your to-do items, so you can access them even if you close your browser or shut down your computer.

Task List 2023 with local storage is built using HTML, CSS, and JavaScript, with the help of the Bootstrap framework for styling.
